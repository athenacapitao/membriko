# Investigação EPDM: Coberturas Planas - Técnicas de Instalação e Detalhes Construtivos

## 1. Introdução

As coberturas planas representam uma das aplicações mais comuns e críticas para membranas EPDM em Portugal. Este documento apresenta uma análise detalhada das técnicas de instalação, detalhes construtivos e considerações específicas para o clima português, com base em normas técnicas, estudos de caso e experiência prática.

---

## 2. Definição e Classificação de Coberturas Planas

### 2.1 Definição Técnica

Segundo o Regulamento das Características de Comportamento Térmico dos Edifícios (RCCTE):

- **Cobertura plana**: Inclinação ≤ 10% (≈5.7°)
- **Cobertura inclinada**: Inclinação > 10%

### 2.2 Tipos de Coberturas Planas

| Tipo | Inclinação | Aplicação |
|------|------------|-----------|
| **Tradicional** | 1-3% | Edifícios residenciais |
| **Invertida** | 1-3% | Com isolamento sobre membrana |
| **Acessível** | 1-2% | Terraços transitáveis |
| **Verde** | 1-3% | Com vegetação |
| **Técnica** | 0.5-1% | Equipamentos no topo |

### 2.3 Considerações para Portugal

| Região | Consideração Específica |
|--------|-------------------------|
| **Norte** | Drenagem intensa (chuva >1500mm/ano) |
| **Centro** | Amplitude térmica significativa |
| **Lisboa** | Vento, poluição atmosférica |
| **Algarve** | Radiação UV intensa, calor |
| **Ilhas** | Vento, salinidade |

## 3. Sistemas de Instalação de EPDM em Coberturas Planas

### 3.1 Sistema Totalmente Aderido (Fully Adhered)

#### 3.1.1 Descrição

A membrana é colada diretamente ao substrato com adesivo especial.

**Vantagens:**
- Excelente resistência ao vento
- Sem movimento da membrana
- Detalhes precisos
- Ideal para formas complexas

**Desvantagens:**
- Custo mais elevado
- Requer substrato perfeito
- Temperatura de instalação crítica

#### 3.1.2 Especificações Técnicas

| Parâmetro | Requisito | Norma |
|-----------|-----------|-------|
| **Adesivo** | Específico para EPDM | EN 13967 |
| **Consumo** | 300-400 g/m² | Fabricante |
| **Temperatura** | +5°C a +35°C | EN 13416 |
| **Humidade substrato** | ≤ 4% | EN 13813 |

#### 3.1.3 Processo de Instalação

```
1. PREPARAÇÃO DO SUBSTRATO
   - Limpeza completa
   - Regularização (tolerância ±5mm/2m)
   - Secagem (≤4% humidade)

2. APLICAÇÃO DO ADESIVO
   - Rolo de lã de aço (tipo mohair)
   - Espalhamento uniforme
   - Tempo de abertura: 10-30 minutos

3. COLOCAÇÃO DA MEMBRANA
   - Alinhamento preciso
   - Pressão com rolo de borracha
   - Eliminação de bolhas de ar

4. SOLDADURA DAS JUNTAS
   - Limpeza com solvente apropriado
   - Aplicação de fita de soldadura
   - Pressão com rolo de aço
```

### 3.2 Sistema Mecanicamente Fixado (Mechanically Fastened)

#### 3.2.1 Descrição

A membrana é fixada com placas e parafusos, com sobreposição soldada.

**Vantagens:**
- Instalação rápida
- Menor custo
- Adequado para grandes áreas

**Desvantagens:**
- Menor resistência ao vento
- Requer estrutura adequada
- Pontos de fixação visíveis

#### 3.2.2 Especificações Técnicas

| Parâmetro | Requisito | Norma |
|-----------|-----------|-------|
| **Placas** | Aço galvanizado | EN 10147 |
| **Parafusos** | Aço inox A2/A4 | EN ISO 3506 |
| **Densidade** | 4-6 placas/m² | Projeto |
| **Comprimento** | ≥ penetração 25mm | EN 14509 |

#### 3.2.3 Padrões de Fixação

| Zona | Densidade | Observações |
|------|-----------|-------------|
| **Interior** | 4 placas/m² | Áreas protegidas |
| **Bordos** | 6 placas/m² | 1m da borda |
| **Cantos** | 8 placas/m² | Refuerzo especial |
| **Áreas vento** | +50% | Zonas expostas |

### 3.3 Sistema Solto com Lastro (Ballasted)

#### 3.3.1 Descrição

A membrana é colocada solta e lastrada com cascalho ou pavimento.

**Vantagens:**
- Instalação mais simples
- Proteção mecânica adicional
- Isolamento térmico

**Desvantagens:**
- Peso adicional significativo
- Não adequado para inclinações >5%
- Manutenção mais difícil

#### 3.3.2 Especificações Técnicas

| Parâmetro | Requisito | Norma |
|-----------|-----------|-------|
| **Lastro** | Cascalho 20-40mm | EN 13242 |
| **Peso** | 50-80 kg/m² | Cálculo vento |
| **Geotêxtil** | Proteção da membrana | EN 13249 |
| **Drenagem** | Camada de drenagem | EN 13252 |

### 3.4 Comparação de Sistemas para Portugal

| Critério | Aderido | Mecânico | Lastrado |
|----------|---------|----------|----------|
| **Custo (€/m²)** | 25-35 | 20-30 | 18-28 |
| **Instalação** | Difícil | Média | Fácil |
| **Vento** | Excelente | Bom | Regular |
| **Manutenção** | Fácil | Média | Difícil |
| **Vida útil** | 40-50 anos | 40-50 anos | 40-50 anos |
| **Recomendação** | Alta | Média | Baixa |

## 4. Detalhes Construtivos Críticos

### 4.1 Bordos e Terminações

#### 4.1.1 Terminação em Parede

```
DETALHE TÉCNICO - TERMINAÇÃO EM PAREDE

1. Altura mínima: 150mm acima da cobertura
2. Reforço: Fita de reforço 300mm
3. Fixação: Perfil de alumínio anodizado
4. Selante: Silicone neutro de alta qualidade
5. Proteção: Chapéu de alumínio ou chapa zincada
```

**Especificações:**
- Perfil: Alumínio 6063 T5, anodizado 25µ
- Fixação: Parafusos inox A4, 200mm centros
- Selante: Silicone neutro, classe 25LM

#### 4.1.2 Terminação em Caleira

```
DETALHE TÉCNICO - TERMINAÇÃO EM CALEIRA

1. Extensão: 100mm dentro da caleira
2. Inclinação: ≥ 2% para drenagem
3. Fixação: Perfil de canto
4. Impermeabilização: Fita líquida complementar
5. Drenagem: Saída mínima Ø100mm
```

### 4.2 Penetrações

#### 4.2.1 Tubos e Condutas

| Diâmetro | Solução | Detalhe |
|----------|---------|---------|
| **< 100mm** | Manga EPDM | Soldada à membrana |
| **100-300mm** | Colarinho metálico | Selante compressível |
| **> 300mm** | Estrutura elevada | Detalhe especial |

#### 4.2.2 Saídas de Ar Condicionado

```
DETALHE TÉCNICO - SAÍDA AR CONDICIONADO

1. Base: Placa de apoio em aço galvanizado
2. Impermeabilização: Manga EPDM soldada
3. Fixação: Parafusos inox com arruelas
4. Selante: Massa butílica
5. Proteção: Colarinho metálico
```

### 4.3 Juntas de Dilatação

#### 4.3.1 Juntas Estruturais

| Movimento | Solução | Detalhe |
|-----------|---------|---------|
| **< 25mm** | Fita de soldadura reforçada | 300mm cada lado |
| **25-50mm** | Perfil de dilatação EPDM | Fixação mecânica |
| **> 50mm** | Ponte móvel | Projeto especial |

#### 4.3.2 Cálculo de Dilatação

**Fórmula:**
```
ΔL = L × α × ΔT
Onde:
ΔL = Dilatação (mm)
L = Comprimento (mm)
α = 160×10⁻⁶/°C (EPDM)
ΔT = Variação temperatura (°C)
```

**Exemplo Portugal (Lisboa):**
- Comprimento: 20m = 20,000mm
- ΔT: 45°C (-5°C a +40°C)
- ΔL = 20,000 × 160×10⁻⁶ × 45 = 144mm

### 4.4 Áreas Críticas

#### 4.4.1 Cantos Internos

```
SOLUÇÃO PARA CANTOS INTERNOS

1. Reforço: Fita de reforço triangular
2. Dimensões: 300×300mm
3. Aplicação: Antes da membrana principal
4. Soldadura: Perímetro completo
5. Pressão: Rolo de aço
```

#### 4.4.2 Mudanças de Plano

| Ângulo | Solução | Observações |
|--------|---------|-------------|
| **< 90°** | Reforço pré-moldado | Fabricante |
| **90°** | Reforço triangular | 300mm cada lado |
| **> 90°** | Reforço especial | Projeto |

## 5. Preparação do Substrato

### 5.1 Tipos de Substrato

| Tipo | Preparação | Observações |
|------|------------|-------------|
| **Betão** | Regularização, cura completa | Humidade ≤4% |
| **Madeira** | Placa OSB/contraplacado | Fixação adequada |
| **Metal** | Limpeza, primário | Compatibilidade |
| **Isolamento** | Placas coladas/parafusadas | Estabilidade |

### 5.2 Requisitos de Planeidade

| Parâmetro | Tolerância | Método |
|-----------|------------|--------|
| **Desnível** | ±5mm/2m | Régua 2m |
| **Ondulação** | ≤3mm/300mm | Régua 300mm |
| **Superfície** | Lisa, sem saliências | Inspeção visual |

### 5.3 Controlo de Humidade

| Método | Limite | Equipamento |
|--------|--------|-------------|
| **Hygrometria** | ≤4% | Higrómetro de perfuração |
| **Plástico** | Sem condensação | Método ASTM D4263 |
| **CM** | ≤75% | Higrómetro ambiente |

## 8. Manutenção Preventiva

### 8.1 Programa de Manutenção

| Frequência | Ação | Observações |
|------------|------|-------------|
| **Anual** | Inspeção visual | Primavera |
| **Bianual** | Limpeza drenagens | Outono/Primavera |
| **5 anos** | Ensaio estanquidade | Áreas críticas |
| **10 anos** | Avaliação completa | Especialista |

### 8.2 Checklist de Inspeção

```
CHECKLIST ANUAL - COBERTURA PLANA EPDM

1. DRENAGEM
   [ ] Saídas desobstruídas
   [ ] Caleiras limpas
   [ ] Inclinação mantida

2. MEMBRANA
   [ ] Sem rasgos ou perfurações
   [ ] Soldaduras intactas
   [ ] Sem bolhas ou descolamentos

3. DETALHES
   [ ] Terminações firmes
   [ ] Penetrações estanques
   [ ] Juntas sem fissuras

4. ACESSÓRIOS
   [ ] Lastro uniforme
   [ ] Proteções firmes
   [ ] Sinalização visível
```

## 9. Estudos de Caso em Portugal

### 9.1 Caso 1: Edifício Residencial - Porto

| Parâmetro | Valor |
|-----------|-------|
| **Ano** | 2018 |
| **Área** | 850m² |
| **Sistema** | Totalmente aderido |
| **Espessura** | 1.5mm |
| **Resultado** | Sem infiltrações após 6 anos |

**Lições aprendidas:**
- Importância da inclinação mínima (2%)
- Reforço extra em áreas vento
- Manutenção anual essencial

| **Sistema** | Mecanicamente fixado |
| **Espessura** | 2.0mm |
| **Resultado** | Excelente performance |

**Lições aprendidas:**
- Densidade de fixação crítica
