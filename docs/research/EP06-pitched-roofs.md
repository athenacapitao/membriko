# Investigação EPDM: Telhados Inclinados - Sistemas e Detalhes Construtivos

## 1. Introdução

Os telhados inclinados representam uma aplicação especializada de membranas EPDM que requer considerações técnicas específicas devido à inclinação, escoamento de águas e fixação mecânica. Este documento apresenta uma análise detalhada dos sistemas EPDM para telhados inclinados, com foco nas condições climáticas portuguesas e nos requisitos normativos nacionais.

---

## 2. Definição e Classificação de Telhados Inclinados

### 2.1 Definição Técnica

Segundo o Regulamento das Características de Comportamento Térmico dos Edifícios (RCCTE):

- **Telhado inclinado**: Inclinação > 10% (≈5.7°)
- **Telhado muito inclinado**: Inclinação > 70% (≈35°)

### 2.2 Classificação por Inclinação

| Categoria | Inclinação | Aplicação Típica |
|-----------|------------|------------------|
| **Baixa inclinação** | 10-25% | Coberturas industriais |
| **Média inclinação** | 25-50% | Habitação unifamiliar |
| **Alta inclinação** | 50-70% | Arquitetura tradicional |
| **Muito inclinado** | >70% | Reabilitação histórica |

### 2.3 Limites de Aplicação EPDM

| Fabricante | Inclinação Máxima | Sistema |
|------------|------------------|---------|
| **Carlisle** | 100% (45°) | Totalmente aderido |
| **Firestone** | 83% (40°) | Mecanicamente fixado |
| **Johns Manville** | 100% (45°) | Especial inclinado |
| **Soprema** | 70% (35°) | Sistema Portugal |

## 3. Sistemas de Instalação para Telhados Inclinados

### 3.1 Sistema Totalmente Aderido para Inclinações

#### 3.1.1 Princípio de Funcionamento

A membrana é colada diretamente ao substrato com adesivo de alta resistência, criando uma ligação integral que resiste à gravidade e ao vento.

**Vantagens específicas para inclinações:**
- Eliminação do risco de deslizamento
- Estanquidade absoluta
- Adaptação a formas complexas
- Resistência a ventos fortes

**Limitações:**
- Requer substrato perfeito
- Temperatura de instalação crítica
- Custo mais elevado

#### 3.1.2 Especificações Técnicas para Inclinações

| Parâmetro | Inclinação < 30% | Inclinação 30-50% | Inclinação > 50% |
|-----------|------------------|-------------------|------------------|
| **Adesivo** | Standard | Alta aderência | Especial inclinado |
| **Consumo** | 350 g/m² | 400 g/m² | 450 g/m² |
| **Tempo abertura** | 20 min | 15 min | 10 min |
| **Pressão aplicação** | 5 kg | 8 kg | 10 kg |

#### 3.1.3 Processo de Instalação para Inclinações

```
PROCESSO ESPECÍFICO PARA TELHADOS INCLINADOS

1. PREPARAÇÃO DO SUBSTRATO
   - Limpeza abrasiva (jateamento se necessário)
   - Regularização (tolerância ±3mm/2m)
   - Secagem completa (≤3% humidade)
   - Aplicação de primário (se necessário)

2. SEQUÊNCIA DE INSTALAÇÃO
   - Trabalhar de baixo para cima
   - Faixas horizontais de 1-2m de altura
   - Sobreposição mínima 150mm
   - Pressão imediata com rolo pesado

3. CONTROLO DE QUALIDADE
   - Ensaio pull-off após 24h
   - Verificação de aderência total
   - Inspeção de soldaduras
```

### 3.2 Sistema Mecanicamente Fixado para Inclinações

#### 3.2.1 Princípio de Funcionamento

A membrana é fixada com placas e parafusos especiais para inclinações, com sistema de retenção adicional.

**Vantagens:**
- Instalação mais rápida
- Menor dependência das condições climáticas
- Adequado para grandes superfícies

**Desvantagens:**
- Maior densidade de fixação necessária
- Pontos de fixação críticos
- Requer estrutura adequada

#### 3.2.2 Especificações de Fixação para Inclinações

| Inclinação | Densidade Placas | Tipo Placa | Comprimento Parafuso |
|------------|------------------|------------|---------------------|
| **10-25%** | 6 placas/m² | Standard | 75mm |
| **25-50%** | 8 placas/m² | Reforçada | 100mm |
| **>50%** | 10 placas/m² | Especial inclinado | 125mm |

#### 3.2.3 Padrões de Fixação

```
PADRÃO DE FIXAÇÃO PARA TELHADOS INCLINADOS

1. ZONA INFERIOR (primeiros 2m)
   - Densidade: +50% do padrão
   - Tipo: Placas de retenção
   - Verificação: Ensaio pull-out

2. ZONA MÉDIA
   - Densidade: Padrão conforme inclinação
   - Alinhamento: Linhas horizontais
   - Espaçamento: 300-400mm

3. ZONA SUPERIOR
   - Densidade: Padrão
   - Fixação extra: Bordo superior
   - Proteção: Chapéu de terminação
```

### 3.3 Sistema de Retenção Mecânica

#### 3.3.1 Perfis de Retenção

| Tipo | Material | Aplicação |
|------|----------|-----------|
| **Perfil contínuo** | Alumínio anodizado | Bordo inferior |
| **Barra transversal** | Aço galvanizado | Zonas intermediárias |
| **Ganchos específicos** | Aço inox A4 | Pontos críticos |

#### 3.3.2 Cálculo de Forças

**Fórmula para cálculo de retenção:**
```
F = W × sen(θ) + Fv
Onde:
F = Força total de retenção (N/m)
W = Peso da membrana (N/m²)
θ = Ângulo de inclinação (°)
Fv = Força do vento (N/m²)
```

**Exemplo para Portugal:**
- Peso EPDM: 2 kg/m² = 19.6 N/m²
- Inclinação: 30° (58% inclinação)
- Vento: 120 km/h = 800 N/m²
- F = 19.6 × sen(30°) + 800 = 9.8 + 800 = 809.8 N/m²

## 4. Detalhes Construtivos Específicos para Inclinações

### 4.1 Cumeeiras e Espigões

#### 4.1.1 Cumeeira (Ridge)

```
DETALHE TÉCNICO - CUMEEIRA EPDM

1. PREPARAÇÃO
   - Reforço triangular 300×300mm
   - Aplicação antes da membrana principal
   - Soldadura perimetral completa

2. INSTALAÇÃO
   - Membrana contínua sobre a cumeeira
   - Sobreposição mínima 200mm cada lado
   - Fixação com perfil de alumínio

3. PROTEÇÃO
   - Chapéu de alumínio anodizado
   - Fixação com parafusos inox
   - Selante silicone neutro
```

#### 4.1.2 Espigão (Hip)

```
DETALHE TÉCNICO - ESPIGÃO EPDM

1. GEOMETRIA
   - Corte em bisel 45°
   - Sobreposição mínima 150mm
   - Reforço adicional 400mm

2. INSTALAÇÃO
   - Trabalhar do inferior para superior
   - Pressão com rolo de aço
   - Verificação de estanquidade

3. PROTEÇÃO
   - Perfil específico para espigão
   - Fixação mecânica adicional
   - Selante complementar
```

### 4.2 Rincões (Valleys)

#### 4.2.1 Rincão Aberto

| Parâmetro | Requisito | Observações |
|-----------|-----------|-------------|
| **Largura mínima** | 300mm | Para drenagem adequada |
| **Inclinação** | ≥ 5% | Prevenção de estagnação |
| **Reforço** | Dupla camada | Área crítica |
| **Proteção** | Grade metálica | Contra detritos |

#### 4.2.2 Rincão Fechado

```
DETALHE TÉCNICO - RINCÃO FECHADO

1. PREPARAÇÃO
   - Limpeza completa da área
   - Aplicação de primário
   - Reforço pré-moldado

2. INSTALAÇÃO
   - Membrana contínua através do rincão
   - Sem juntas na linha do rincão
   - Pressão com rolo de borracha

3. VERIFICAÇÃO
   - Ensaio de estanquidade
   - Inspeção visual completa
   - Documentação fotográfica
```

### 4.3 Beirais e Aleros

#### 4.3.1 Beiral (Eave)

| Elemento | Especificação | Função |
|----------|---------------|--------|
| **Extensão** | 300-600mm | Proteção da fachada |
| **Inclinação** | 5-10% | Drenagem eficiente |
| **Goteiro** | Perfil contínuo | Coleta de águas |
| **Proteção** | Grade anti-pássaros | Prevenção de ninhos |

#### 4.3.2 Alero (Soffit)

```
DETALHE TÉCNICO - ALERO VENTILADO

1. VENTILAÇÃO
   - Área livre ≥ 500 cm²/m
   - Proteção contra insetos
   - Acesso para manutenção

2. IMPERMEABILIZAÇÃO
   - Extensão da membrana 100mm
   - Selante de transição
   - Proteção contra UV

3. ACABAMENTO
   - Perfil de alumínio
   - Fixação oculta
   - Compatibilidade estética
```

### 4.4 Penetrações em Telhados Inclinados

#### 4.4.1 Chaminés

| Tipo | Solução | Detalhe |
|------|---------|---------|
| **Chaminé central** | Manga EPDM | Soldada à membrana |
| **Chaminé de bordo** | Colarinho metálico | Selante compressível |
| **Chaminé múltipla** | Estrutura elevada | Detalhe especial |

#### 4.4.2 Ventilações

```
DETALHE TÉCNICO - VENTILAÇÃO DE TELHADO

1. LOCALIZAÇÃO
   - Evitar linha de rincão
   - Distância mínima 300mm de detalhes

2. INSTALAÇÃO
   - Base pré-moldada EPDM
   - Soldadura perimetral completa
   - Selante de alta temperatura

3. PROTEÇÃO
   - Grelha anti-pássaros
   - Filtro anti-insetos
   - Manutenção acessível
```

## 5. Subtelhas e Barreiras para Telhados Inclinados

### 5.1 Funções da Subtelha

| Função | Requisito | Produto Típico |
|--------|-----------|----------------|
| **Barreira de água** | EN 13859-1 | Não tecido bituminoso |
| **Barreira de vapor** | Sd ≥ 5m | Membrana aluminizada |
| **Resistência ao vento** | Classe A/B/C | Reforçada com tecido |

### 5.2 Sistemas de Subtelha para EPDM

#### 5.2.1 Sistema de Dupla Proteção

```
SISTEMA DUPLA PROTEÇÃO PARA INCLINAÇÕES ELEVADAS

1. CAMADA INFERIOR (sobre ripado)
   - Subtelha de alta resistência
   - Fixação mecânica completa
   - Sobreposição ≥ 150mm

2. CAMADA SUPERIOR (sob EPDM)
   - Geotêxtil de separação
   - Permeabilidade controlada
   - Compatibilidade com adesivo

3. ESPAÇO DE VENTILAÇÃO
   - Espessura ≥ 40mm
   - Entrada/saída adequada
   - Proteção contra insetos
```

#### 5.2.2 Sistema de Ventilação Integrada

| Componente | Função | Especificação |
|------------|--------|---------------|
| **Ripado** | Suporte | Madeira tratada 25×50mm |
| **Contra-ripado** | Ventilação | Madeira tratada 25×50mm |
| **Subtelha** | Proteção | EN 13859 classe A |
| **Ventilação** | Fluxo de ar | Área ≥ 300 cm²/m |

### 5.3 Considerações para Portugal
